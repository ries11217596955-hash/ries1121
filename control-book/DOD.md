# Definition of Done (DoD)

Обязательные условия готовности для любого внешнего релиза:
1. RU/AZ‑парность разделов: заголовки и ключевые блоки дублированы.
2. ISO‑структура 4–10 соблюдена; подсказки включены.
3. Подписи: требуемое число ролей (см. Signature Policy Matrix).
4. Линтеры пройдены: Vale/markdownlint; орфография без критических ошибок.
5. SemVer‑тег установлен; CHANGELOG обновлён.
6. Release Playbook выполнен: README, LICENSE, превью, MANIFEST, SHA256.
7. PDF/A прилагается; при веб-публикации — PDF/UA + WCAG 2.2 чек.
8. QA‑Gate: Gate‑1/2 OK; протокол дефектов приложен (если был).
