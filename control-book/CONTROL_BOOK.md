# Control Book (Источник истины)

Этот документ фиксирует правила фабрики: режимы, гейты, роли, ритмы, артефакты.
- Режим: PATCH‑ONLY по умолчанию (меняем только явно заданное).
- Гейты: Smart Audit / DesignGate / PreviewGate / Answer↔Reality.
- Артефакты: DoD, QA‑SHEET, PATCH‑LOG, RELEASE PLAYBOOK, TERMBASE/STYLE.
- Роли: Owner, Project Lead, DocOps Lead, QA Lead, vSME, PMM, MI.
- Ритм: Weekly Status; Decision Memo для решений >60 мин или >100$.
